O Que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas na web. Ele permite definir a estrutura de um documento web através de elementos e tags que delimitam o conteúdo.

Estrutura Básica de um Documento HTML

Um documento HTML começa com a declaração <!DOCTYPE html> que define o tipo de documento. A estrutura básica é composta pelas seguintes tags: 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é o meu primeiro documento HTML.</p>
</body>
</html>

Elementos e Tags HTML

Tags são usadas para marcar o início e o fim de um elemento. A maioria das tags tem uma tag de abertura <tag> e uma tag de fechamento </tag>. Alguns elementos comuns incluem:

  • <h1> a <h6>: Títulos de diferentes níveis.
  • <p>: Parágrafo.
  • <a>: Link.
  • <img>: Imagem.
  • <ul>, <ol>, <li>: Listas não ordenadas e ordenadas.

Atributos de Tags

Tags podem ter atributos que fornecem informações adicionais sobre o elemento. Atributos são definidos na tag de abertura e consistem em um nome e um valor, como href em uma tag de link: 

<a href="https://www.example.com">Visite o Exemplo</a>

Formatação de Texto

HTML permite a formatação de texto com diversas tags, tais como:

  • <b> ou <strong>: Texto em negrito.
  • <i> ou <em>: Texto em itálico.
  • <u>: Texto sublinhado.

Listas

Listas podem ser não ordenadas ou ordenadas. Elas são compostas por itens de lista (<li>).

Lista Não Ordenada:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Lista Ordenada:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Links e Imagens

Links e imagens são elementos essenciais em HTML.

Link:

<a href="https://www.example.com">Clique aqui</a>

Imagem:

<img src="imagem.jpg" alt="Descrição da imagem">

Tabelas

Tabelas são usadas para exibir dados tabulares.

<table>
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Maria</td>
        <td>30</td>
    </tr>
</table>

Formulários

Formulários permitem a entrada de dados pelo usuário.

<form action="/enviar-dados" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Enviar">
</form>

Comentários

Comentários em HTML são ignorados pelo navegador e são usados para deixar notas no código. 

<!-- Este é um comentário -->

Boas Práticas

  • Usar tags semânticas: Tags como <header>, <footer>, <article>, e <section> melhoram a acessibilidade e SEO.
  • Validar o código: Ferramentas como o W3C Validator ajudam a garantir que o HTML está correto.
  • Manter um código limpo e organizado: Facilita a manutenção e compreensão do código.

Conclusão

HTML é a base da criação de páginas web. Compreender os conceitos básicos é o primeiro passo para se tornar um desenvolvedor web. Praticar a criação de páginas simples e explorar diferentes elementos e atributos é fundamental para o aprendizado.
